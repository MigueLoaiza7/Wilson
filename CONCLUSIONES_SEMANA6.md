# Conclusiones Semana 6

## 31. ¿Qué comandos Git se usaron con mayor frecuencia?

git add, git commit, git push, git pull, git status y git branch.

## 32. ¿Qué diferencia se encontró entre trabajo local y remoto?

El trabajo local se realiza en el computador del desarrollador, mientras que el remoto se almacena y comparte mediante GitHub.

## 33. ¿Qué parte del flujo con ramas resultó más clara o más difícil?

La creación de ramas fue sencilla; la parte que requiere más atención es la fusión de cambios mediante merge.

## 34. ¿Qué valor tienen Git y GitHub para un proyecto DevOps?

Permiten mantener trazabilidad, colaboración, control de versiones y sirven como base para procesos de integración y despliegue continuo.
